# ADHD Game Simulation: "Save the Muse Home" Analysis

## Source
Nature Translational Psychiatry - "A mobile device-based game prototype for ADHD: development and preliminary feasibility testing"
https://www.nature.com/articles/s41398-024-02964-2

## Game Overview
**"Save the Muse Home (SMH)"** is a serious game designed to alleviate ADHD symptoms through cognitive training. The game features a fantasy storyline about a warrior striving to save his homeland, with integrated training scenes targeting multiple cognitive domains.

## Core Cognitive Domains Targeted
1. **Executive Inhibition** - Controlling impulses and suppressing unwanted responses
2. **Motor Inhibition** - Controlling physical responses and movements
3. **Sustained Attention** - Maintaining focus over extended periods
4. **Selective Attention** - Focusing on relevant stimuli while ignoring distractions
5. **Working Memory** - Holding and manipulating information temporarily
6. **Planning** - Organizing actions to achieve goals

## Game Scenes & Mechanics

### Scene 1: Treasure Chest Identification
**Cognitive Focus:** Sustained Attention + Working Memory
- **Mechanic:** Players identify real gems among changing treasure chests
- **Challenge:** Continuous focused attention required
- **Skill:** Ability to track multiple targets simultaneously
- **Difficulty:** Chests change/transform, requiring constant attention

### Scene 2: Reward-Redemption System
**Cognitive Focus:** Planning + Executive Function
- **Mechanic:** Players earn gems from training scenes
- **Goal:** Save enough gems to exchange for pets (advanced rewards)
- **Challenge:** Planning required to complete the main character's mission
- **Motivation:** Dual reward system (fixed + variable ratio schemes)

### Scene 3: Anti-Saccade Task
**Cognitive Focus:** Executive Inhibition + Working Memory
- **Mechanic:** Central fixation point → target dot appears at 4 positions (3°, 6° left/right)
- **Task:** Inhibit impulse to look at stimulus, calculate opposite position, make volitional saccade
- **Duration:** 20 trials (1000ms per position, 300ms feedback)
- **Measures:** Inhibitory control and spatial working memory

### Scene 4: Delayed-Saccade Task
**Cognitive Focus:** Inhibitory Control + Spatial Working Memory
- **Mechanic:** Fixation point remains, green dot appears (100ms), then blank screen (1-5s delay)
- **Task:** Recall original position of stimulus when blank screen appears
- **Measures:** Early saccades (inhibitory control), position recall accuracy (working memory)
- **Difficulty:** Variable delays (1s, 3s, 5s)

### Scene 5: Continuous Performance Test (CPT)
**Cognitive Focus:** Sustained Attention + Impulse Control
- **Mechanic:** Rapid stimulus presentation (go/no-go paradigm)
- **Task:** Click for "go" stimuli (square below box), refrain for "no-go" stimuli
- **Duration:** 504 trials total (2 sets of 252 trials)
- **Measures:** Reaction time, accuracy, commission errors, sensitivity index (d')

## Gamification Elements

### Reward System
- **Fixed Ratio Rewards:** Consistent rewards for completing tasks
- **Variable Ratio Rewards:** Unpredictable rewards to maintain engagement
- **Progression:** Gems → Pet exchanges → Mission completion

### Narrative & Engagement
- **Storyline:** Fantasy theme with empathetic character
- **Responsiveness:** Game responds to player actions
- **Progression:** Successive narratives that advance with player performance
- **Moderate Challenge:** Difficulty balanced to maintain engagement

## Game Design Philosophy
1. **Multi-domain Training:** Integrates all 6 cognitive domains into one cohesive game
2. **Child-Friendly:** Illustrated with children-friendly images
3. **Portable:** Designed for mobile and tablet devices
4. **Therapeutic:** Grounded in neuroplasticity and cognitive training principles
5. **Engaging:** Combines serious cognitive training with entertaining gameplay

## Study Results (Preliminary)
- **ADHD Group:** Significant improvements in CPT accuracy, reaction time, anti-saccade, delayed-saccade, and SNAP-IV scores
- **Neurotypical Group:** Minimal significant changes
- **Engagement:** Strong interest in games across both groups
- **Parent Feedback:** ADHD parents reported perceived benefits and willingness to continue

## Implementation Focus for Simulation
1. Treasure chest identification game (visual attention + working memory)
2. Anti-saccade task simulator (eye movement tracking or cursor-based)
3. Delayed-saccade task simulator
4. Continuous Performance Test (go/no-go)
5. Reward system with gem collection
6. Progress tracking and scoring
7. Fantasy-themed UI with engaging narrative elements
